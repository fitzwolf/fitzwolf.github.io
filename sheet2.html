About the data

The data collected for this project was done using a simple python script to grab the ticker information starting from December 03, 2007 (start of the Great Recession) until January 1, 2023. The tickers chosen for this were AAPL, MSFT (Microsoft), SPY (an etf that closely mirrors S&P 500 performance) and SLV (an etf that follows the performance of silver). All of the data was downloaded on July 25, 2023.

Code used: 


                import yfinance as yf
                import pandas as pd
                
                try:
                    # Download historical data as a pandas DataFrame
                    data = yf.download('MSFT', start='2007-12-01', end='2023-01-01')
                
                    # If the DataFrame is empty, raise an exception
                    if data.empty:
                        raise ValueError('No data downloaded')
                
                    # Print the first few rows
                    print(data.head())
                
                    # Save the data to a CSV file
                    data.to_csv('data/msft_stock_data.csv')
                
                except Exception as e:
                    print(f'An error occurred: {e}')
